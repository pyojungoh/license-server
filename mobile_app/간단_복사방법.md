# 가장 간단한 파일 복사 방법

## 방법 1: 파일 탐색기에서 직접 복사 (추천)

### 1. Android Studio 프로젝트 위치 찾기
- Android Studio에서 프로젝트 열기
- 상단 메뉴: **File → Open**
- 프로젝트가 있는 폴더 경로 확인 (예: `C:\Users\사용자명\AndroidStudioProjects\HanjinAutomation`)

### 2. 파일 복사
1. 파일 탐색기 열기
2. 두 폴더를 나란히 열기:
   - 왼쪽: `C:\hanjin\mobile_app\android\app\src\main\java\com\hanjin\`
   - 오른쪽: `[프로젝트경로]\app\src\main\java\com\`

3. 왼쪽의 `hanjin` 폴더 전체를 복사 → 오른쪽에 붙여넣기
   - `automation` 폴더가 있다면 삭제하거나 덮어쓰기

4. 리소스 파일 복사:
   - `C:\hanjin\mobile_app\android\app\src\main\res\` 폴더 내용
   - → `[프로젝트경로]\app\src\main\res\` 폴더에 붙여넣기

5. 설정 파일들 복사:
   - `C:\hanjin\mobile_app\android\app\src\main\AndroidManifest.xml`
   - `C:\hanjin\mobile_app\android\app\build.gradle.kts`
   - `C:\hanjin\mobile_app\android\build.gradle.kts`
   - `C:\hanjin\mobile_app\android\settings.gradle.kts`
   - → 각각 해당 위치에 복사

### 3. Android Studio에서 새로고침
- Android Studio로 돌아가기
- 왼쪽 파일 트리에서 `app` 폴더 오른쪽 클릭 → **Synchronize 'app'**
- 또는 상단 메뉴: **File → Sync Project with Gradle Files**

### 4. 패키지 이름 확인
- 파일 트리에서 `LoginActivity.kt` 파일 열기
- 파일 맨 위에 `package com.hanjin` 또는 `package com.hanjin.automation` 확인
- 모든 파일이 동일한 패키지를 사용하도록 통일

## 방법 2: Android Studio에서 직접 만들기

복사가 복잡하면 Android Studio에서 직접 파일을 만들고 내용을 복사:

1. 파일 트리에서 `app/src/main/java/com/hanjin/` 폴더 만들기 (없으면)
2. 각 파일을 만들고 내용 복사:
   - `LoginActivity.kt` → 내용 복사
   - `MainActivity.kt` → 내용 복사
   - 등등...
