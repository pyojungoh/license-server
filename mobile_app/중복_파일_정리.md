# 중복 파일 정리 방법

## 문제
MainActivity가 두 곳에 있어서 오류 발생:
- `com.example.hanjinautomation.MainActivity` (Android Studio가 생성한 기본 파일)
- `com.hanjin.MainActivity` (우리가 만든 파일)

## 해결 방법

### 방법 1: com.example 폴더 삭제 (추천)

1. 왼쪽 파일 트리에서 `app/src/main/java/com/example/` 폴더 찾기
2. `example` 폴더를 우클릭 → Delete
3. 확인 창이 뜨면 "OK" 클릭

### 방법 2: 파일만 삭제

1. `app/src/main/java/com/example/hanjinautomation/MainActivity.kt` 파일 찾기
2. 파일 우클릭 → Delete
3. 확인 창이 뜨면 "OK" 클릭

## 확인

정리 후에는 다음 구조만 남아야 합니다:
```
app/src/main/java/com/
└── hanjin/
    ├── Config.kt
    ├── LoginActivity.kt
    ├── MainActivity.kt  ← 이 파일만 있어야 함
    ├── api/
    ├── ble/
    ├── models/
    └── utils/
```

`com.example` 폴더는 삭제해도 됩니다.
