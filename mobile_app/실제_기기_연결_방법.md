# 실제 Android 기기에 앱 설치하기

## 방법 1: USB 케이블 연결 (권장)

### 1단계: 모바일 기기에서 개발자 옵션 활성화

1. **설정** 앱 열기
2. **휴대전화 정보** 또는 **디바이스 정보** 찾기
3. **빌드 번호**를 7번 연속으로 터치
   - "개발자가 되었습니다!" 메시지가 표시됨

### 2단계: USB 디버깅 활성화

1. **설정** → **개발자 옵션** (또는 **시스템** → **개발자 옵션**)
2. **USB 디버깅** 토글 켜기
3. 확인 메시지에서 **확인** 클릭

### 3단계: USB 케이블로 연결

1. Android 기기를 USB 케이블로 PC에 연결
2. 모바일 화면에 "USB 디버깅을 허용하시겠습니까?" 팝업이 나타남
3. **이 컴퓨터에서 항상 허용** 체크박스 선택
4. **허용** 클릭

### 4단계: Android Studio에서 실행

1. Android Studio에서 상단 툴바의 실행 버튼(▶) 클릭
2. 또는 `Shift + F10` (Windows/Linux) 또는 `Ctrl + R` (Mac)
3. **실행 대상 선택** 창이 나타나면 연결된 기기 선택
4. **OK** 클릭
5. 앱이 자동으로 빌드되어 기기에 설치됨

## 방법 2: WiFi로 연결 (USB 없이)

### 사전 준비
- Android 11 이상
- PC와 기기가 같은 WiFi 네트워크에 연결

### 1단계: USB로 초기 연결 (한 번만)

1. USB 케이블로 연결 (위의 방법 1 참고)
2. Android Studio 터미널에서:
   ```bash
   adb tcpip 5555
   ```

### 2단계: 기기의 IP 주소 확인

1. **설정** → **WiFi**
2. 연결된 WiFi 옆의 ⓘ 아이콘 터치
3. **IP 주소** 확인 (예: 192.168.0.100)

### 3단계: WiFi로 연결

Android Studio 터미널에서:
```bash
adb connect [기기IP주소]:5555
```
예: `adb connect 192.168.0.100:5555`

### 4단계: Android Studio에서 실행

- 방법 1과 동일하게 실행 버튼 클릭

## 방법 3: APK 파일로 직접 설치

### 1단계: APK 파일 생성

1. Android Studio에서 **Build** → **Build Bundle(s) / APK(s)** → **Build APK(s)**
2. 빌드 완료 메시지가 나타나면 **locate** 클릭
3. 또는 `app/build/outputs/apk/debug/app-debug.apk` 경로에서 파일 찾기

### 2단계: 모바일로 전송

- USB 케이블로 복사
- 이메일로 전송
- 클라우드 스토리지 사용
- 메신저로 전송

### 3단계: 모바일에서 설치

1. APK 파일 클릭
2. **알 수 없는 소스에서 설치 허용** 팝업이 나타나면:
   - **설정** 클릭 → **이 출처 허용** 토글 켜기
3. **설치** 클릭
4. 설치 완료!

## 연결 확인 방법

Android Studio 하단의 **Device Manager** 또는 터미널에서:
```bash
adb devices
```

연결된 기기가 표시되면 성공:
```
List of devices attached
ABC123XYZ    device
```

## 문제 해결

### 기기가 인식되지 않는 경우

1. USB 디버깅이 켜져 있는지 확인
2. USB 케이블을 다른 포트에 연결
3. USB 드라이버 설치 (제조사 웹사이트)
4. 기기 재부팅
5. PC 재부팅

### "앱 설치 실패" 오류

1. 기기의 저장 공간 확인
2. 이전 버전 앱 삭제 후 재설치
3. **설정** → **앱** → **특수 앱 액세스** → **앱 설치** 에서 허용 확인
