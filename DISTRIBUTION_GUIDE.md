# 프로그램 배포 및 판매 가이드

## 📦 배포 준비 단계

### 1. 실행 파일 빌드

#### 1.1 PyInstaller 설치
```bash
pip install pyinstaller
```

#### 1.2 실행 파일 생성
```bash
python build_exe.py
```

생성된 파일: `dist/송장번호일괄처리시스템.exe`

#### 1.3 테스트
- 다른 PC에서 실행 파일 테스트
- 필요한 DLL/라이브러리 확인
- 바이러스 백신 오탐지 확인 (필요시 코드 서명)

### 2. 패키징 구성

#### 2.1 배포 패키지 구조
```
배포패키지/
├── 송장번호일괄처리시스템.exe  # 메인 프로그램
├── ESP32_설치가이드.pdf        # 하드웨어 설정 가이드
├── 사용자매뉴얼.pdf            # 프로그램 사용법
├── 라이선스등록가이드.pdf      # 라이선스 등록 방법
└── README.txt                  # 빠른 시작 가이드
```

#### 2.2 필수 포함 사항
- 실행 파일
- ESP32 하드웨어 설정 가이드
- 프로그램 사용 매뉴얼
- 라이선스 등록 방법 안내
- FAQ (자주 묻는 질문)

### 3. 라이선스 시스템 개선

#### 3.1 현재 시스템
- 로컬 파일 기반 라이선스 검증
- 만료일 기반 제한

#### 3.2 개선 옵션

**옵션 A: 온라인 인증 (권장)**
- 서버에서 라이선스 키 검증
- 하드웨어 ID 기반 바인딩
- 실시간 만료일 확인
- 장점: 보안 강화, 복사 방지
- 단점: 인터넷 연결 필요, 서버 운영 비용

**옵션 B: 오프라인 라이선스 (현재)**내가
- 로컬 파일 기반 검증
- 암호화된 라이선스 키https://github.com/pyojungoh/license-server.git
- 장점: 인터넷 불필요, 간단
- 단점: 보안 취약, 복사 가

**옵션 C: 하이브리드**
- 온라인 활성화 (1회)
- 이후 오프라인 사용 가능
- 주기적 온라인 검증 (선택)

### 4. 가격 정책 제안

#### 4.1 패키지 구성
1. **기본 패키지** (예: 50,000원)
   - 프로그램 라이선스 (1년)
   - ESP32 하드웨어 (1개)
   - 설치 가이드
   - 사용자 매뉴얼

2. **확장 패키지** (예: 80,000원)
   - 프로그램 라이선스 (무제한)
   - ESP32 하드웨어 (1개)
   - 우선 기술 지원
   - 업데이트 무료 제공

3. **기업 패키지** (예: 200,000원)
   - 다중 라이선스 (5개)
   - ESP32 하드웨어 (5개)
   - 전담 기술 지원
   - 맞춤 기능 개발

### 5. 판매 채널

#### 5.1 온라인 판매
- **네이버 스마트스토어**: 간편한 개인 판매
- **쿠팡**: 높은 노출, 배송 대행
- **11번가**: 추가 판매 채널
- **직접 판매**: 개인 홈페이지/블로그

#### 5.2 오프라인 판매
- 택배 업체 직접 방문
- 물류 박람회 참가
- 지역 택배 협회 협력

#### 5.3 마케팅 전략
- **타겟 고객**: 한진택배 지점, 물류센터, 택배 대행 업체
- **홍보 방법**:
  - 유튜브 데모 영상
  - 블로그 후기
  - 택배 관련 커뮤니티 홍보
  - 직접 방문 판매
