# 빠른 판매 시작 가이드

## 🚀 3단계로 시작하기

### Step 1: 실행 파일 만들기 (30분)

1. **PyInstaller 설치**
   ```bash
   pip install pyinstaller
   ```

2. **실행 파일 빌드**
   ```bash
   python build_exe.py
   ```

3. **테스트**
   - `dist/송장번호일괄처리시스템.exe` 실행
   - 다른 PC에서도 테스트

### Step 2: 라이선스 키 생성 (5분)

1. **라이선스 생성**
   ```bash
   python LICENSE_GENERATOR.py
   ```

2. **고객에게 제공**
   - 생성된 `licenses/license_XXXXXX.txt` 파일 전달
   - 또는 라이선스 키 직접 전달

### Step 3: 판매 시작

#### 방법 A: 직접 판매 (가장 간단)
1. 지인/알고 있는 택배 기사에게 연락
2. 데모 보여주기
3. 계좌 이체로 결제
4. 프로그램 + ESP32 배송 또는 직접 전달

#### 방법 B: 온라인 판매
1. 네이버 스마트스토어 개설
2. 제품 등록
3. 배송 준비

## 💡 첫 고객 확보 팁

### 1. 무료 체험 제공
- 7일 무료 라이선스 제공

### 2. 직접 방문
- 지역 택배 지점 방문
